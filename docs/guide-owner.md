# 👔 Инструкция для владельца

Руководство по использованию админ-панели чек-листа магазина.

---

## Вход в систему

1. Откройте ссылку на приложение в браузере
2. Нажмите кнопку **🔑 Владелец**
3. Введите PIN-код: `0000` (или тот, который вы установили)
4. Вы попадёте в админ-панель

> 💡 После первого входа приложение запомнит вас. В следующий раз вход будет автоматическим.

---

## Главный экран админ-панели

На главном экране вы видите:

### Выбор даты
- **Сегодня** — текущий день
- **Вчера** — предыдущий день
- **Календарь** — выбрать любую дату

### Карточки сотрудников

Для каждого сотрудника (Агнеса и Оксана) показано:

- **До 10:00** — прогресс утренних задач (0–9)
- **В теч. дня** — прогресс дневных задач (0–2)
- **До 20:00** — прогресс вечерних задач (0–8)
- **Общий прогресс** — общее количество (0–19)
- **Невыполненные задачи** — список того, что ещё не сделано

### Цвета прогресс-баров

| Цвет | Значение |
|---|---|
| 🟢 Зелёный | Все задачи секции выполнены |
| 🟠 Оранжевый | Часть задач выполнена |
| ⚪ Серый | Ничего не выполнено |

---

## Детальный просмотр

Нажмите на карточку сотрудника, чтобы увидеть все задачи:

- ✅ — выполнено вовремя (с именем и временем)
- ⚠️ — выполнено с опозданием
- ⬜ — не выполнено

## Журнал «Покупатели зашли»

Внизу страницы — журнал нажатий кнопки «Покупатели зашли — телефоны убраны». Показывает кто и во сколько нажал.

## Telegram-уведомления

Вы будете автоматически получать уведомления:

| Событие | Пример |
|---|---|
| Задача выполнена | ✅ Агнеса отметила: "Открыть дверь" |
| Задача с опозданием | ⚠️ Оксана отметила с опозданием: "Музыка" |
| Секция завершена | 🎉 Агнеса завершила утренний чек-лист! |
| Дедлайн пропущен | ❌ Утренний чек-лист не завершён до 10:00 |
| Напоминание | ⏰ До 10:00 осталось 10 минут |

## Изменение PIN-кодов

1. Откройте Google Таблицу «Чек-лист магазина»
2. Перейдите на лист **config**
3. Измените значения:
   - `pin_agnesa` — PIN Агнесы
   - `pin_oksana` — PIN Оксаны
   - `pin_owner` — ваш PIN
4. Также обновите PIN в файле `js/config.js` в репозитории GitHub

---

## Просмотр данных в Google Таблице

Вы можете просматривать данные напрямую в Google Таблице:

- **Лист tasks** — все отметки задач (дата, сотрудник, задача, время, опоздание)
- **Лист customers_log** — все нажатия кнопки «Покупатели зашли»
- **Лист config** — настройки системы

---

## Добавление/изменение задач

Если нужно изменить список задач:

1. Откройте файл `js/config.js` в репозитории GitHub
2. Найдите массив `TASKS`
3. Добавьте, измените или удалите задачи
4. Также обновите код в файле `google-apps-script/Code.gs` (массивы `TASK_COUNTS` и `TASK_NAMES`)
5. В Apps Script создайте **новое развёртывание**

## Решение проблем

| Проблема | Что делать |
|---|---|
| Не вижу данные в админке | Обновите страницу (потяните вниз на телефоне) |
| Данные не обновляются | Автообновление каждые 2 минуты. Обновите вручную |
| Telegram не приходит | Проверьте настройки в [setup-telegram.md](setup-telegram.md) |
| Сотрудник забыл PIN | Откройте лист config в Google Sheets |
| Нужно сбросить день | Удалите строки за этот день в листе tasks |
