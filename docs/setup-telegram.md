# 🤖 Настройка Telegram-бота

Пошаговая инструкция по созданию Telegram-бота для уведомлений.

---

## Шаг 1. Создать бота

1. Откройте Telegram
2. Найдите бота **@BotFather** (официальный бот от Telegram)
3. Отправьте команду: `/newbot`
4. Введите имя бота: `Чек-лист Магазина Бот`
5. Введите username бота: `checklist_your_store_bot`
   - Username должен заканчиваться на `bot`
   - Должен быть уникальным
6. BotFather ответит и выдаст **токен** вида:
   ```
   7123456789:AAHBk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```
7. **Скопируйте этот токен** — он понадобится дальше

> ⚠️ Никому не показывайте токен! Он даёт полный доступ к боту.

---

## Шаг 2. Получить Chat ID владельца

Чтобы бот мог отправлять сообщения, нужен Chat ID получателя.

### Вариант А: Через @userinfobot

1. Найдите бота **@userinfobot** в Telegram
2. Нажмите **Start**
3. Бот покажет ваш **Id** — это и есть chat_id
4. Скопируйте число (например: `123456789`)

### Вариант Б: Через API

1. Сначала отправьте **любое сообщение** вашему новому боту в Telegram
2. Откройте в браузере:
   ```
   https://api.telegram.org/bot7123456789:AAHBk-xxxx/getUpdates
   ```
   (замените токен на ваш)
3. Найдите в ответе:
   ```json
   "chat": { "id": 123456789 }
   ```
4. Число `123456789` — это ваш chat_id

---

## Шаг 3. Записать в Google Sheets

1. Откройте вашу Google Таблицу «Чек-лист магазина»
2. Перейдите на лист **config**
3. Заполните:

| key | value |
|---|---|
| telegram_bot_token | `7123456789:AAHBk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxx` |
| telegram_chat_id | `123456789` |

> 💡 Значения вставляйте **без кавычек**.

---

## Шаг 4. Протестировать

1. Откройте Google Apps Script (из Google Таблицы → Расширения → Apps Script)
2. Выберите функцию `testTelegram` в выпадающем списке
3. Нажмите **▶ Выполнить**
4. Проверьте — в Telegram должно прийти тестовое сообщение:
   ```
   🧪 Тестовое сообщение из чек-листа магазина!
   Если вы видите это — Telegram настроен верно.
   ```

---

## Групповой чат (опционально)

Если хотите получать уведомления в группу:

1. Создайте группу в Telegram
2. Добавьте бота в группу
3. Отправьте любое сообщение в группу
4. Откройте:
   ```
   https://api.telegram.org/botВАШ_ТОКЕН/getUpdates
   ```
5. Найдите `chat_id` группы (обычно отрицательное число, например `-987654321`)
6. Запишите этот ID в лист config

После настройки вы будете получать такие сообщения:

🕐 Время: 09:38

```

🕐 Время: 10:17
```

```
🎉 Агнеса завершила Утренний чек-лист!
🕐 Время: 09:55

```

```

```

## Решение проблем

| Проблема | Решение |
|---|---|
| Бот не отправляет | Проверьте токен и chat_id в листе config |
| Ошибка «Chat not found» | Отправьте сообщение боту первым, потом попробуйте снова |
| Уведомления в группу не приходят | Убедитесь что бот добавлен в группу как участник |
| Двойные уведомления | Проверьте что нет дублирующих триггеров в Apps Script |
